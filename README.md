# CardCraft - Business Card Creator

A modern, beautiful business card creator built with React, TypeScript, and Tailwind CSS. Create stunning business cards with multiple templates, color themes, and export them as high-quality PNG images. Features a smooth onboarding flow, 3D card effects, confetti animations, and PWA capabilities.

## ✨ Features

- **🎨 Multiple Templates**: Modern, Gradient, Minimal, and Neon designs
- **🌈 Color Themes**: Cyan, Purple, Pink, and Green color schemes
- **👁️ Live Preview**: Real-time preview as you customize your card
- **📤 Export Options**: Download as high-quality PNG images
- **📱 Sharing**: Share cards via native sharing or copy to clipboard
- **📱 PWA Support**: Install as a native app on mobile devices
- **📱 Responsive Design**: Works perfectly on desktop and mobile
- **💾 Local Storage**: All cards saved locally in your browser
- **🎉 Confetti Effects**: Celebratory animations on save/share
- **🔄 3D Card Effects**: Interactive tilt effects on hover
- **👆 Swipe Animations**: Smooth gesture-based navigation
- **🎭 Smooth Onboarding**: Step-by-step card creation flow
- **⚡ Offline Support**: Works without internet connection

## 🚀 Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/bowei/cardcraft.git
cd cardcraft
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:3000`

### Alternative Ports

You can also run the app on different ports:
```bash
npm run dev:8080  # Run on port 8080
npm run dev:4000  # Run on port 4000
```

### Build for Production

```bash
npm run build
npm run preview
```

## 🎯 Usage

1. **🎨 Create New Card**: Click "Create New Card" to start the onboarding flow
2. **📋 Choose Template**: Select from Modern, Gradient, Minimal, or Neon designs
3. **🌈 Pick Colors**: Choose your preferred color theme (Cyan, Purple, Pink, Green)
4. **📝 Add Information**: Fill in your personal and contact details
5. **👁️ Preview**: See your card update in real-time as you type
6. **💾 Save**: Save your card to your collection with confetti celebration
7. **📤 Export**: Download as high-quality PNG or share with others
8. **✏️ Edit**: Modify existing cards anytime
9. **🗑️ Manage**: Delete cards you no longer need

## 🚀 Deployment

### Deploy to Vercel (Recommended)

The easiest way to deploy CardCraft is using Vercel:

1. **Push to GitHub**:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/yourusername/cardcraft.git
git push -u origin main
```

2. **Deploy on Vercel**:
   - Go to [vercel.com](https://vercel.com)
   - Click "New Project"
   - Import your GitHub repository
   - Click "Deploy"

### Alternative Deployment Options

- **Netlify**: Drag and drop the `dist` folder
- **GitHub Pages**: Use GitHub Actions for automatic deployment
- **Firebase Hosting**: Use Firebase CLI for deployment

### Configuration Files

- `vercel.json` - Optimized for PWA and SPA routing
- `manifest.json` - PWA configuration
- `sw.js` - Service worker for offline support

For detailed deployment instructions, see [DEPLOYMENT.md](DEPLOYMENT.md).
